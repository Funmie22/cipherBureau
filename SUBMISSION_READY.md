## 📞 Support

### If Playtest Doesn't Work
1. Install Devvit CLI: `npm install -g devvit@latest`
2. Authenticate: `devvit auth`
3. Try again: `npm run dev`

### If ZIP Won't Upload
1. Verify file: `ls -lah dist/cipher-bureau-devvit.zip`
2. Ensure ZIP created: `npm run pack:devvit`
3. Check devvit.yaml for indentation errors

### If App Won't Load in Devvit
1. Check for TypeScript errors: `npx tsc --noEmit`
2. Verify webview built: `ls webview/dist/index.html`
3. Check console for errors: Open browser DevTools (F12)
